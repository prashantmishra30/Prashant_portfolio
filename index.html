AI / Healthcare
01 / PROJECT

Myopia Detection using AI

Deep Learning — Retinal Fundus Image Diagnosis

An automated diagnostic pipeline for Pathological Myopia (PM) — a severe structural eye disease and a leading cause of blindness. Manual screening of retinal fundus images is labour-intensive and requires high expertise. This project addresses that with a high-throughput ResNet-18 model trained on 100,000+ retinal images, enabling objective and rapid early-stage myopia screening.

Engineering Highlights: Managed 54 GB of raw data via the Kaggle REST API with a custom preprocessing pipeline (ImageNet normalisation, 224×224 standardisation, tensor conversion). Optimised data loading with Pinned Memory and multi-threaded DataLoaders. Leveraged transfer learning on a pre-trained ResNet-18 backbone with Adam Optimiser and Cross-Entropy Loss, trained on a 15 GB VRAM Tesla T4 GPU across 10 epochs. Model validated with Confusion Matrices — prioritising Recall to minimise false negatives in clinical settings — and tested on independent datasets to ensure generalisation.

Python PyTorch ResNet-18 OpenCV NumPy / Pandas Scikit-learn Google Colab (T4 GPU) Kaggle API
